Xtense pour OGSpy - Development Build

Xtense est un Script développé par l'OGSteam permettant de récupérer les informations de notre Jeu préféré.

Le Mod permet de récupérer l'ensemble des informations du Jeu, tels que les rapports d'espionnages, les galaxies, les rapports de combat, etc
