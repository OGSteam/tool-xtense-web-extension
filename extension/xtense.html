Xtense pour OGSpy

Xtense est un Script développé par l'OGSteam permettant de récupérer les informations de notre Jeu préféré.

Le Mod permet de récupérer l'ensemble des informations du Jeu, tels que les rapports d'espionnages, les galaxies, les rapports de combat, etc

Configuration du Mod

Nous avons réalisé une documentation sur le wiki afin de vous aider dans la configuration.

Cliquez sur ce lien pour acceder à la documentation.

La configuration est également accessible dans le mod Xtense de votre serveur OGSpy.
